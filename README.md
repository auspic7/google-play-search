# Google Play Search

아름다운 UI로 Google Play 앱을 검색하세요!

## 기능
- 🔍 키워드 검색
- 👨‍💻 개발자 ID 검색
- 🇰🇷 한국어/영어 지원
- 📊 상세 정보 (평점, 설치 수, 출시일, 업데이트일)

## 로컬 실행
```bash
docker compose up -d
```

브라우저: http://localhost:3000

## Coolify 배포
1. Coolify에서 "New Resource" → "Docker Compose" 선택
2. Git 저장소 연결
3. Port: 3000
4. Deploy!
